03 · Main MCP Server
Main MCP Orchestrator
🗂️ Tool Registry
Discovers and indexes all tools exposed by registered domain MCPs. Serves unified tools/list manifest to clients. Hot-reload on domain registration.
🔀 Intelligent Router
Routes each tools/call to the correct domain server by tool namespace prefix (e.g. perf.*, aum.*). Supports fan-out for cross-domain queries.
📦 Response Aggregator
Merges parallel responses from multiple domains into a single structured reply. Handles partial failures gracefully with fallback stubs.
💾 Result Cache
TTL-based caching of read-only tool results (Redis). Per-tool TTL config. Cache bypass header for real-time requests. Cache key = tool + args hash.
🩺 Health Monitor
Polls each domain MCP /healthz every 10 s. Circuit-breaker pattern: open after 3 failures, half-open probe at 30 s. Degrades gracefully.
🔐 Secret Manager
Holds upstream credentials for all domain servers. Rotates secrets via Vault or AWS Secrets Manager. Domain servers never hold long-lived credentials.
📋 Schema Validator
Validates all inbound tool arguments against JSON Schema before dispatch. Rejects malformed calls early; protects domain servers from bad input.
🔌 Domain Registry
Service-discovery sidecar. Domain MCPs self-register on startup with name, endpoint, capabilities, and SLA tier. Deregistered on graceful shutdown.
